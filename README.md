# 🌐 Site AcessaNet

Este projeto é o site institucional da **AcessaNet**, um provedor de internet fibra óptica.  
O objetivo do projeto é apresentar os serviços da empresa, planos de internet, benefícios e facilitar o contato com clientes.

## 🚀 Funcionalidades
- Apresentação dos planos de internet
- Informações sobre a empresa
- Benefícios inclusos nos planos
- Área de contato
- Layout moderno e responsivo
- Integração com redes sociais

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- React
- Git e GitHub para versionamento

## 🎯 Objetivo
Criar uma plataforma moderna, rápida e intuitiva para apresentar os serviços da AcessaNet e facilitar a aquisição de planos de internet.
